--Installations
1. Install the latest version of Node.js-include 'npm' with the install

--Project
1. Open the project folder in your code editor (we used VSCode)
2. Navigate to the root/parent folder of the project. 
3. Run the following command: npm install . This will install the necessary packages for the project
4. Then do 'cd/server' in the terminal to move into the 'server' folder. Run the following command: npm install . 
    This will install the necessary packages for the backend
5. Move to the client folder using the following command: cd ../client . Perform 'npm install' in the client folder
    This will install the necessary packages for the front end

--Run the project
1. In the root folder, perform the following command: npm start
    This will start the server and launch the "landing" page of the Project
